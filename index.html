ANIMEX

Description :

Animex est un projet d'application de streaming d'anime japonais. Ce projet est basé sur le SDK Flutter de Google. Il devra pouvoir:

  • Lire une video à partir d'un lien
  • Récuperer les informations d'un anime sur une BDD ou un fichier JSON
  • Permettre de telecharger les episodes d'un anime
  • S'adapter aux utilisateurs

Début :

Novembre 2020

Fin :

En developpement...

Technologies utilisées :

Dart via le SDK Flutter

SYADOX

Description :

Le site SYADOX est un site communautaire crée pour le youtubeur Syadox. Sur ce site One-Page on peut retrouver :

  • Une brève présentation du youtubeur
  • Un carousel vidéo regroupant les vidéos de sa chaîne
  • Un formulaire de Contact
  • Et bientôt : un formulaire d'inscription et une partie blog

Début :

Mars 2020

Fin :

Mai 2020 ( ajout de nouvelles fonctionnalités encore à ce jour )

Technologies utilisées :

HTML5, CSS3, Javascript
